# 🎬 CineDine

An all-in-one movie booking platform that allows users to book tickets, get personalized movie recommendations, and pre-order food for a seamless cinema experience 🍿

# 🚀 Features

🎟️ ## Movie Ticket Booking
Browse movies and show timings
Select seats in real-time
Book tickets instantly

## 🤖 Smart Recommendations
Get movie suggestions based on preferences
Trending & popular movies section

## 🍔 Food Ordering Integration
Pre-order snacks before reaching the theatre
Avoid queues and save time

## 👤 User Authentication
Secure login/signup
User profile & booking history

## 💳 Payment Integration
Smooth and secure checkout
